CityofGits - 3D Developer Portfolio Platform (2022)

Developed in 2022 during Grade 9 (U.S. K12 Grade 8) at Ananda College, CityofGits is a web application designed to solve the problem of static, 2D developer portfolios. I identified that many of the current portfolios developers use are very static - not inherently an issue or a market gap, but I recognized a lack of engaging options especially for teenagers and people who get into programming early. They seek a kind of engagement that traditional portfolios cannot provide. For this project, I used Three.js, which was challenging to learn at that time. I built a 3D world where developers and creators can construct personalized, interactive portfolio spaces.

Each user can create a unique link (e.g., cityofgits.link/neksha) that directs visitors to their custom-built 3D environment. The platform supports both public profiles and password-protected private portfolios.

One special feature is the commenting system - anyone can leave appropriate comments on the main city page. When someone creates a new portfolio page, it is automatically announced in the chat, and users can also add custom messages. This creates a community-driven experience for developers showcasing their work.

The primary technical challenge was managing the significant server-side resources required to store and render each user's 3D world state. This led me to explore and adopt Cloudflare Workers and Pages for scalable deployment - experience that later proved invaluable for subsequent projects like Litha 2024. Originally, this project was hosted on a free subdomain (cityofgits.netlify.app), but in 2025 when I was able to acquire the domain cityofgits.link, I transferred it there. At that point, I integrated this project into my nonprofit, Open Rockets Foundation, and handed over maintenance to Avin Indrasoma, a Google Developer Expert and GitHub Campus Expert (https://github.com/avincodes).

Tempfy - Bluetooth-Enabled Digital Thermometer (2022)

Developed and presented at the Impreza Competition in 2022 at Ananda College, Tempfy is an Arduino-based digital thermometer with Bluetooth Low Energy (BLE) wireless transmission. The device uses a DHT11 temperature sensor to measure ambient temperature, displays readings on a 16x2 I2C LCD screen, and broadcasts the data via BLE using an nRF24 radio module. The system categorizes temperature as "cold" (below 27°C), "normal" (27-33°C), or "warm" (above 33°C) and includes error handling for sensor disconnection.
