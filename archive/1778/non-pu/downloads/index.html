Requirements
Windows 10 / 11   Required
64-bit only. The application will not run on 32-bit systems or earlier Windows versions.
CH340 USB Serial Driver   Required for device use
Required to communicate with connected hardware over USB. Download and run the installer below, then restart if prompted.
CH340 Driver (ch341ser.inf) Right-click the .inf file → Install
LAV Filters   Optional — for video codec
Required only if you use the video capture or NVD codec features. Download from the official LAV Filters release page.
LAV Filters on GitHub
Getting Started
  1. Download nonpu-instant.exe using the button above.
  2. If you are connecting a hardware device via USB, install the CH340 driver first. Right-click ch341ser.inf and select Install. Restart your computer if prompted.
  3. Run nonpu-instant.exe directly. No installation is required.
  4. On first launch, follow the on-screen license activation steps.
